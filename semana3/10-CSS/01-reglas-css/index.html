Reglas CSS

¿Qué es la sintaxis CSS?

Se considera sintaxis al orden y relación de los elementos que componen una correcta instrucción.

¿Qué es una regla CSS?

Una regla CSS es la forma adecuada de escribir una instrucción mediante código CSS, es decir la manera en que se debe escribir código CSS para que este sea interpretado y aplicado correctamente por el navegador web.

Partes de una regla CSS


        selector { 

          propiedad: valor; (declaración)

        }

Una regla CSS está formada principalmente por un selector y una declaración, la declaración siempre va dentro de llaves y dentro se encuentran las propiedades con su respectivo valor.

  • El selector: es quien indica a que elemento HTML se aplica el código CSS (el estilo).
  • La propiedad: es la característica que se desea definir, por ejemplo: color (el color).
  • Valor: es el valor que tomará la propiedad, por ejemplo: green (verde).
  • Punto y coma (;): una declaración siempre se debe terminar con un punto y coma (;)

Tipos de Selectores

Selector Universal

El selector universal es el selector más general, es decir, se aplica a todos los elementos HTML.


        * { 

          propiedad: valor;

          ...
        }

Selector de tipo

Los selectores de tipo en CSS se refieren a los selectores que apuntan a un tipo de elemento HTML como puede ser:


        h1 { 

          propiedad: valor;

          ...
        }

Selector de clase

Este tipo de selectores en CSS apuntan a un elemento HTML identificado, es decir a un elemento HTML que tenga el atributo class.


        .nombreDeLaClase { 

          propiedad: valor;

          ...
        }

El selector de clase es una forma de identificar un elemento HTML, es decir, un elemento HTML que tenga una clase.

Selector con identidicador (id)

Tanto el selector de tipo id y class funcionan de la misma manera; la única diferencia es que para los selectores de tipo id se le antepone el signo de numeral (#) y a los de tipo class un punto (.)


        #identificadorUnico { 

          propiedad: valor;

          ...
        }

Selector de atributo

El selector de atributos CSS coincide con los elementos en función de la presencia o el valor de un atributo determinado.

Ejemplo: 


        /* Elementos <a> con un atributo title */

        a[title] {

          propiedad: valor;

          ...
        }

        

        /* Elementos <a> con un href que coincida con "https://example.org" */

        a[href="https://example.org"] {

          propiedad: valor;

          ...
        }

        

        /* Elementos <a> con un href que contenga "example" */

        a[href*="example"] {

          propiedad: valor;

          ...
        }

        

        /* Elementos <a> con un href que comience con "#" */

        a[href^="#"] {

          propiedad: valor;

          ...
        }

        

        /* Elementos <a> con un href que termine en ".org" */

        a[href$=".org"] {

          propiedad: valor;

          ...
        }

        

        /* Elementos <a> cuyo atributo class contenga la palabra "logo" */

        a[class~="logo"] {

          propiedad: valor;

          ...
        }

Operadores de combinación de selectores

Hijos directos

El combinador > separa a dos selectores y busca solo a los elementos que coindicen con el segundo selector y que son hijos directos del primero.


        .contenedor__principal > p { 

          propiedad: valor;

          ...
        }

Decendientes

El ␣ combinador (que se supone que representan un espacio, o mejor dicho uno o más espacios en blanco) combina dos selectores tales que el selector combinado incluye sólo los elementos que coinciden con el segundo selector para los que hay un elemento ancestro que coincide con el primer selector.


        .contenedor__principal p { 

          propiedad: valor;

          ...
        }

Hermanos adyacentes

Se hace referencia a este selector como selector adyacente o selector del próximo hermano. Sólo seleccionará un elemento especificado que esté inmediatamente después de otro elemento especificado.


        .lista li + li { 

          propiedad: valor;

          ...
        }

Hermanos general

El combinador ~ separa dos selectores y selecciona el segundo elemento sólo si está precedido por el primero y ambos comparten un padre común.


        .parrafo2 span ~ span { 

          propiedad: valor;

          ...
        }

Pseudo Elementos

Al igual que las pseudo classes, los pseudo elementos se añaden a los selectores, pero en cambio, no describen un estado especial sino que, permiten añadir estilos a una parte concreta del documento.


        a[target="_blank"]::before { 

          propiedad: valor;

          ...
        }
