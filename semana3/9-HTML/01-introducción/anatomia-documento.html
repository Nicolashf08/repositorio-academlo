Anatomía de un documento HTML


    <!DOCTYPE html>

    <html lang="en">

    <head>

      <meta charset="UTF-8">

      <meta name="viewport" content="width=device-width, initial-scale=1.0">

      <title>Titulo de la página actual</title>

      <meta name="description" content="No mas de 165 caracteres">

      <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

    </head>

    <body>

    </body>

    </html>

Aquí tenemos:

  • <!DOCTYPE html>: Declaración del tipo de documento es una instrucción especial que va al inicio de nuestro documento HTML y que permite al navegador entender qué versión de HTML estamos utilizando.
  • <html>: Representa la raiz de un documento HTML (root element). El resto de elementos descienden de este elemento.
  • lang="en": Los elementos también pueden tener atributos. Los atributos contienen información extra sobre el elemento que no se mostrará en el contenido. En este caso, el atributo lang sirve para definir el lenguaje en que están escritos los elementos o el lenguaje en el cual deberían de estar escritos.
  • <head>: Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos. No es visible en la página
  • <meta> Meta de "metainformation", Sirve para aportar información sobre el documento. Los atributos debe indicarlo el autor, aunque los mas comunes son name nombre al que se asocia la metainformación y content los datos que se quieren asociar a name.
  • charset="UTF-8" Especifica el tipo de caracteres que se utilizan en el documento.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> Especifica el tamaño de la ventana del navegador. Importante para hacer nuestras páginas responsivas
  • <title>: Este establece el título de la página, que es el título que aparece en la pestaña del navegador en la que se carga la página. El título de la página también se utiliza para describir la página cuando se marca como favorita. No más de 55 y 65 caracteres.
  • <body>: Contiene todo el contenido que quieres mostrar a los usuarios cuando visitan tu página, ya sea texto, imágenes, vídeos, juegos, pistas de audio reproducibles o cualquier otra cosa.
  • <meta name="description" content="">: Especifica una descripción del documento. No más de 165 caracteres.
  • <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">: Especifica el icono del documento.
